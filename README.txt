GITHUB PAGES — FLAT FILE VERSION

All website files are intentionally placed in this single folder.
There are no subfolders.

Upload every file in this folder directly to the root of your GitHub repository.
The file index.html must remain at the repository root.

GitHub Pages settings:
- Source: Deploy from a branch
- Branch: main
- Folder: /(root)

Do not upload only index.html; upload all PDF, PNG, and WEBP files too.
